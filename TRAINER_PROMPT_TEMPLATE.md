# 🎯 Шаблон промпта для анализа тренировок продаж

## Описание системы

Этот промпт предназначен для анализа тренировок продаж, где:
- **N-минутная встреча** разбивается на **N отдельных тренировок**
- Каждая тренировка анализируется **по ролям** (клиент/продавец)
- Оценивается **техническое выполнение** и **эмоциональное воздействие**
- Проверяются **ожидаемые результаты** после тренировки

## Структура промпта

### 1. РАЗБИВКА НА ТРЕНИРОВКИ
```
Критерии разделения:
- Смена темы/продукта
- Пауза в разговоре >30 секунд  
- Переход к новой ситуации/сценарию
- Изменение настроения/тона
```

### 2. ОПРЕДЕЛЕНИЕ РОЛЕЙ
```
Для каждой тренировки:
- Кто менеджер (продавец)
- Кто клиент
- Обоснование выбора 2-3 цитатами с таймкодами
```

### 3. АНАЛИЗ ПО ЭТАПАМ ПРОДАЖ
```
А) ТЕХНИЧЕСКОЕ ВЫПОЛНЕНИЕ (1-10 баллов):
- Подготовка к встрече
- Установление контакта и доверия
- Выявление потребностей клиента
- Презентация решения
- Обработка возражений
- Закрытие сделки/следующие шаги

Б) ЭМОЦИОНАЛЬНОЕ ВОЗДЕЙСТВИЕ (1-10 баллов):
- Интерес клиента
- Уровень доверия
- Желание продолжить общение
- Готовность к покупке
```

### 4. ОЖИДАЕМЫЕ РЕЗУЛЬТАТЫ

#### Хорошая тренировка - ответы клиента:
- "Да, я задумался, желание купить увеличилось"
- "Мне стало интересно, хочется узнать больше"
- "Я почувствовал, что это может решить мою проблему"
- "Продавец понял мои потребности"

#### Плохая тренировка - ответы клиента:
- "Мне вообще не хочется общаться, хочется послать нафиг"
- "Я не понял, зачем мне это нужно"
- "Продавец не слушал меня"
- "Слишком агрессивно, отталкивает"

#### Хорошая тренировка - ответы продавца:
- "Я почувствовал, что клиент заинтересовался"
- "Удалось выяснить его потребности"
- "Клиент задавал вопросы о продукте"
- "Есть прогресс в переговорах"

#### Плохая тренировка - ответы продавца:
- "Клиент не слушал меня"
- "Не удалось заинтересовать"
- "Клиент был закрыт"
- "Нужно работать над техникой"

## Формат ответа

```
=== РАЗБИВКА НА ТРЕНИРОВКИ ===
Тренировка 1: [00:00-08:30] - [Название/тема]
Тренировка 2: [08:30-16:45] - [Название/тема]
...

=== АНАЛИЗ ТРЕНИРОВОК ===

**ТРЕНИРОВКА 1: [Название]**
Роли: speaker_1 → manager, speaker_2 → client
Доказательства: «...» [t=00:12–00:18]

Техническое выполнение:
- Подготовка: 8/10 - комментарий
- Установление контакта: 6/10 - комментарий
- Выявление потребностей: 7/10 - комментарий
- Презентация: 5/10 - комментарий
- Обработка возражений: 4/10 - комментарий
- Закрытие: 3/10 - комментарий

Эмоциональное воздействие:
- Интерес клиента: 6/10 - комментарий
- Доверие: 5/10 - комментарий
- Желание общаться: 4/10 - комментарий
- Готовность к покупке: 3/10 - комментарий

Ожидаемые результаты после тренировки:
- Клиент должен сказать: "..." (оценка: 6/10)
- Продавец должен сказать: "..." (оценка: 5/10)

=== ОБЩАЯ ОЦЕНКА ===
Средний балл по технике: X/10
Средний балл по эмоциям: X/10
Общий прогресс: [описание динамики]
Рекомендации: [конкретные советы]
```

## Настройка под ваши нужды

1. **Добавьте свои критерии оценки** в раздел "ТЕХНИЧЕСКОЕ ВЫПОЛНЕНИЕ"
2. **Уточните ожидаемые фразы** в разделе "ОЖИДАЕМЫЕ РЕЗУЛЬТАТЫ"
3. **Настройте балльную систему** под ваши стандарты
4. **Добавьте специфичные для вашей компании** этапы продаж

## Ограничения по размеру

- **GPT-4o** поддерживает до **128,000 токенов** контекста
- Это примерно **100,000 слов** или **200+ страниц** текста
- **Практически нет ограничений** на размер промпта
- Можете добавлять очень подробные инструкции и примеры

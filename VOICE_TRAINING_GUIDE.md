# 🎤 Руководство по голосовой тренировке с ИИ

## Обзор

Система голосовой тренировки с ИИ — это интерактивная платформа для отработки профессиональных навыков в режиме реального времени с помощью искусственного интеллекта.

## ✨ Возможности

### 1. Реалистичные диалоги
- **Голосовое взаимодействие**: Говорите с ИИ-тренером как с реальным человеком
- **Мгновенная обратная связь**: Получайте советы и рекомендации в реальном времени
- **Адаптивный сценарий**: ИИ подстраивается под ваш уровень и стиль общения

### 2. Разнообразные сценарии
- **Продажи**: Техники продаж, работа с возражениями, закрытие сделок
- **Обслуживание клиентов**: Решение проблем, проявление эмпатии, работа с жалобами
- **Переговоры**: Ведение переговоров, поиск компромиссов, win-win решения

### 3. Система прогресса
- **Чеклисты**: Отслеживайте выполнение ключевых этапов
- **Статистика**: Анализируйте свои результаты
- **Достижения**: Получайте награды за успехи

### 4. Интуитивный интерфейс
- **Современный дизайн**: Красивый и понятный интерфейс
- **Адаптивная верстка**: Работает на любых устройствах
- **Темная тема**: Поддержка темной темы (наследуется от проекта)

## 🚀 Быстрый старт

### Запуск тренировки

1. **Перейдите в дашборд**
   - Откройте `/dashboard` в браузере
   - В верхней части страницы найдите секцию "Голосовые тренировки с ИИ"

2. **Выберите сценарий**
   - **Продажи** (💼): 20-30 мин, 5 этапов
   - **Обслуживание** (🤝): 15-25 мин, 4 этапа
   - **Переговоры** (⚖️): 25-35 мин, 4 этапа

3. **Начните тренировку**
   - Нажмите "Начать тренировку" или выберите карточку сценария
   - Разрешите доступ к микрофону
   - Нажмите большую кнопку микрофона и начинайте говорить

## 🎯 Интерфейс тренировки

### Основные элементы

#### 1. Заголовок (Header)
- **Название тренировки**: Текущий сценарий
- **Статус**: Активна / На паузе
- **Таймер**: Время тренировки
- **Участники**: Количество участников

#### 2. Прогресс-бар
- Показывает % выполнения чеклиста
- Обновляется в реальном времени
- Цветной индикатор прогресса

#### 3. Чат с ИИ (левая часть)
- **Сообщения пользователя**: Синие, справа
- **Сообщения ИИ**: Белые, слева
- **Индикаторы**: "ИИ думает...", "ИИ говорит..."
- **Экспорт**: Сохранение транскрипта диалога

#### 4. Панель участников (правая часть)
- **Вы**: Ваша статистика (ответы, баллы)
- **ИИ-тренер**: Статистика тренера (вопросы, советы)
- **Достижения**: Разблокируемые награды

#### 5. Панель управления (нижняя часть)
- **Большая кнопка микрофона**: Начать/остановить запись
- **Пауза**: Приостановить тренировку
- **Завершить**: Завершить и посмотреть результаты
- **Настройки**: Параметры тренировки

#### 6. Чеклист (боковая панель)
- Открывается по нажатию кнопки 📋
- Показывает этапы тренировки
- Отмечайте выполненные пункты галочкой

### WebSocket протокол

#### Клиент → Сервер

```json
// Начало записи
{
  "type": "start_recording",
  "timestamp": 1234567890
}

// Аудио данные
{
  "type": "audio_data",
  "audio_data": "base64_encoded_audio",
  "timestamp": 1234567890
}

// Остановка записи
{
  "type": "stop_recording",
  "timestamp": 1234567890
}
```

#### Сервер → Клиент

```json
// Статус компонентов
{
  "type": "status",
  "components": {
    "vad": "ready",
    "stt": "ready",
    "gpt": "ready",
    "tts": "ready"
  }
}

// Транскрипция речи
{
  "type": "transcript",
  "text": "Распознанный текст"
}

// Чанк ответа ассистента
{
  "type": "assistant_chunk",
  "text": "Часть ответа"
}

// Завершение ответа
{
  "type": "assistant_complete"
}

// Аудио чанк
{
  "type": "audio_chunk",
  "audio_data": "base64_encoded_audio",
  "sample_rate": 16000
}

// Конец аудио
{
  "type": "audio_end"
}

// Ошибка
{
  "type": "error",
  "message": "Описание ошибки"
}
```

### API endpoints

#### GET `/voice-assistant/training`
Отображает страницу голосовой тренировки

#### POST `/voice-assistant/training/create`
Создает новую тренировочную сессию

```json
{
  "topic": "Тренировка продаж",
  "scenario": "sales",
  "difficulty": "medium",
  "duration_minutes": 30
}
```

#### GET `/voice-assistant/training/{session_id}`
Получает информацию о сессии

#### GET `/voice-assistant/training/{session_id}/stats`
Получает статистику сессии

#### DELETE `/voice-assistant/training/{session_id}`
Удаляет сессию

#### GET `/voice-assistant/scenarios`
Получает список доступных сценариев

### WebSocket `/voice-assistant/ws`
Основной endpoint для голосовой коммуникации

## 🎨 Кастомизация

### Темы
Система использует CSS переменные из `styles.css`:
- `--primary`: Основной цвет (синий)
- `--surface`: Цвет поверхности (белый)
- `--text`: Цвет текста (темный)
- `--muted`: Приглушенный цвет
- `--border`: Цвет границ

### Добавление новых сценариев

1. Отредактируйте `voice_assistant/router.py`
2. Добавьте новый сценарий в функцию `get_training_scenarios()`
3. Создайте чеклист для сценария
4. Добавьте карточку на дашборд

Пример:

```python
{
    "id": "new_scenario",
    "name": "Название",
    "description": "Описание",
    "checklist": [
        {
            "category": "Этап 1",
            "items": ["Пункт 1", "Пункт 2"]
        }
    ]
}
```

## 🐛 Устранение неполадок

### Микрофон не работает
1. Проверьте разрешения браузера
2. Убедитесь, что микрофон не используется другим приложением
3. Перезагрузите страницу

### WebSocket не подключается
1. Проверьте, что backend запущен
2. Проверьте консоль браузера на ошибки
3. Убедитесь, что порт не заблокирован

### ИИ не отвечает
1. Проверьте статус компонентов (`/voice-assistant/health`)
2. Проверьте наличие API ключей (OpenAI, ElevenLabs)
3. Проверьте логи сервера

### Аудио не воспроизводится
1. Проверьте громкость браузера
2. Проверьте настройки аудио в настройках тренировки
3. Проверьте, что TTS сервис доступен

## 📝 Примеры использования

### Базовая тренировка

```javascript
// Создание тренировочной сессии
fetch('/voice-assistant/training/create', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        topic: 'Продажи телефонов',
        scenario: 'sales',
        difficulty: 'medium',
        duration_minutes: 30
    })
})
.then(res => res.json())
.then(data => {
    console.log('Сессия создана:', data.session_id);
    // Перейти на страницу тренировки
    window.location.href = `/voice-assistant/training`;
});
```

### Получение сценариев

```javascript
// Получить список сценариев
fetch('/voice-assistant/scenarios')
    .then(res => res.json())
    .then(data => {
        console.log('Доступные сценарии:', data.scenarios);
        data.scenarios.forEach(scenario => {
            console.log(`- ${scenario.name}: ${scenario.description}`);
        });
    });
```

# 🎯 Руководство по системе анализа тренировок продаж

## Обзор системы

Система анализа тренировок продаж позволяет:
- **Разбивать 40-минутные встречи** на отдельные тренировки
- **Анализировать каждую тренировку** по ролям (клиент/продавец)
- **Оценивать техническое выполнение** и эмоциональное воздействие
- **Проверять достижение целевых состояний** по 10-балльной шкале
- **Выдавать техническую оценку** качества проведения

## Как работает система

### 1. Загрузка аудио
- Пользователь загружает аудиофайл в чат тренера
- Система автоматически расшифровывает речь
- Создается JSON-диалог с таймкодами

### 2. Анализ промптом
- ИИ получает промпт из базы данных (`sales_trainer`)
- Анализирует диалог согласно инструкциям в промпте
- Разбивает встречу на отдельные тренировки
- Оценивает каждую тренировку по критериям

### 3. Результат анализа
- Детальный отчет по каждой тренировке
- Балльные оценки по технике и эмоциям
- Ожидаемые результаты для участников
- Общие рекомендации и выводы

## Настройка промпта

### Доступ к редактору
1. Зайдите в **Админ-панель**
2. Перейдите в **Промпты**
3. Нажмите **"Редактировать промпт тренера"**

### Структура промпта
Промпт должен включать:

#### 1. Разбивка на тренировки
```
Критерии разделения:
- Смена темы/продукта
- Пауза в разговоре >30 секунд
- Переход к новой ситуации/сценарию
- Изменение настроения/тона
```

#### 2. Определение ролей
```
Для каждой тренировки:
- Кто менеджер (продавец)
- Кто клиент
- Обоснование выбора цитатами с таймкодами
```

#### 3. Критерии оценки
```
А) ТЕХНИЧЕСКОЕ ВЫПОЛНЕНИЕ (1-10 баллов):
- Подготовка к встрече
- Установление контакта и доверия
- Выявление потребностей клиента
- Презентация решения
- Обработка возражений
- Закрытие сделки/следующие шаги

Б) ЭМОЦИОНАЛЬНОЕ ВОЗДЕЙСТВИЕ (1-10 баллов):
- Интерес клиента
- Уровень доверия
- Желание продолжить общение
- Готовность к покупке
```

#### 4. Ожидаемые результаты
```
Хорошая тренировка - ответы клиента:
- "Да, я задумался, желание купить увеличилось"
- "Мне стало интересно, хочется узнать больше"
- "Я почувствовал, что это может решить мою проблему"

Плохая тренировка - ответы клиента:
- "Мне вообще не хочется общаться, хочется послать нафиг"
- "Я не понял, зачем мне это нужно"
- "Продавец не слушал меня"
```

## Ограничения и возможности

### Размер промпта
- **GPT-4o** поддерживает до **128,000 токенов** контекста
- Это примерно **100,000 слов** или **200+ страниц** текста
- **Практически нет ограничений** на размер промпта
- Можете добавлять очень подробные инструкции

### Точность анализа
- ИИ анализирует только то, что есть в диалоге
- Не придумывает факты и эмоции
- Приводит конкретные цитаты с таймкодами
- Оценивает строго по заданным критериям

## Настройка под ваши нужды

### 1. Добавьте свои критерии
В разделе "ТЕХНИЧЕСКОЕ ВЫПОЛНЕНИЕ" добавьте специфичные для вашей компании этапы:
- Знание конкурентов
- Работа с бюджетом
- Техническая экспертиза
- И т.д.

### 2. Уточните ожидаемые фразы
В разделе "ОЖИДАЕМЫЕ РЕЗУЛЬТАТЫ" добавьте фразы, которые должны говорить ваши участники:
- Специфичные для вашего продукта
- Соответствующие вашей корпоративной культуре
- Отражающие ваши стандарты качества

### 3. Настройте балльную систему
- Измените веса критериев
- Добавьте дополнительные метрики
- Настройте пороговые значения

## Заключение

Система анализа тренировок продаж — это мощный инструмент для:
- **Объективной оценки** качества тренировок
- **Выявления проблемных зон** в работе менеджеров

## 🚀 Как использовать

### 1. Система уведомлений (фронтенд)

```javascript
// Показать уведомление
window.notifications.success('Файл загружен', 'Успех!');
window.notifications.error('Ошибка загрузки', 'Ошибка');
window.notifications.warning('Файл слишком большой', 'Внимание');
```

### 2. Кеширование (backend)

```python
from services.caching_service import cached

@cached('user_data', ttl=300)  # 5 минут
def get_user_data(user_id: int):
    # expensive operation
    return data
```

### 3. Оптимизация БД (backend)

```python
from services.db_optimizer import DBOptimizer

# Пакетная вставка
DBOptimizer.bulk_insert(db, Message, messages_list)

# Очистка старых записей
DBOptimizer.cleanup_old_records(db, Message, 'created_at', days_to_keep=30)
```

### 4. Мониторинг производительности

Админ-панель доступна по адресу: `/admin/performance` (только для администраторов)

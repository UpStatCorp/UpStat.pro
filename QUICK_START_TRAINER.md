✅ **Добавлены новые роуты** для управления промптом тренера  
✅ **Создан базовый промпт** для анализа тренировок  
✅ **Добавлена кнопка** в админ-панель для доступа к редактору  

## Как использовать

### 1. Зайдите в админ-панель
```
http://localhost:8000/admin/prompts
```

### 2. Нажмите "Редактировать промпт тренера"
Оранжевая кнопка с иконкой тренера

### 3. Скопируйте улучшенный промпт
Откройте файл `ENHANCED_TRAINER_PROMPT.md` и скопируйте весь промпт (без заголовков)

### 4. Вставьте в редактор
- В поле "Название версии": `Анализ тренировок продаж v2.0`
- В поле "Описание": `Улучшенный промпт с детальной оценкой по баллам`
- В поле "Промпт": вставьте скопированный текст

### 5. Сохраните
Нажмите "Сохранить новую версию"

### 6. Протестируйте
- Зайдите в чат тренера: `http://localhost:8000/chat_trener`
- Загрузите аудиофайл тренировки
- Получите детальный анализ с разбивкой на тренировки и оценками

## Что получите

### Разбивка на тренировки
```
Тренировка 1: [00:00-08:30] - Установление контакта
Тренировка 2: [08:30-16:45] - Выявление потребностей
Тренировка 3: [16:45-25:20] - Презентация решения
```

### Оценки по баллам
```
Техническое выполнение:
- Подготовка: 8/10
- Установление контакта: 6/10
- Выявление потребностей: 7/10

Эмоциональное воздействие:
- Интерес клиента: 6/10
- Доверие: 5/10
- Готовность к покупке: 3/10
```

### Ожидаемые результаты
```
Клиент должен сказать: "Интересно, но нужно подумать" (оценка: 6/10)
Продавец должен сказать: "Клиент слушал, но не очень активен" (оценка: 5/10)
```

## Настройка под ваши нужды

### 1. Измените критерии оценки
В разделе "ТЕХНИЧЕСКОЕ ВЫПОЛНЕНИЕ" добавьте свои этапы продаж

### 2. Добавьте ожидаемые фразы
В разделе "ОЖИДАЕМЫЕ РЕЗУЛЬТАТЫ" укажите, что должны говорить ваши участники

### 3. Настройте балльную систему
Измените веса критериев под ваши стандарты

## Ограничения

- **Размер промпта**: практически неограничен (до 128,000 токенов)
- **Точность**: зависит от качества аудио и промпта
- **Скорость**: анализ занимает 1-2 минуты

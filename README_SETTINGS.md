## 🚀 Как запустить

### 1. Запуск приложения
```bash
cd app
python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 2. Открыть в браузере
```
http://localhost:8000/settings
```

### 3. Войти в систему
- Используйте существующий аккаунт или зарегистрируйтесь
- Перейдите в раздел "Настройки" в боковом меню

## 🧪 Тестирование функциональности

### 📝 Редактирование профиля
1. В карточке "Основная информация" измените имя, email или телефон
2. Нажмите "Сохранить изменения"
3. Увидите уведомление об успехе

### 🖼️ Загрузка аватара
1. В карточке "Фото профиля" нажмите "Выбрать файл"
2. Выберите изображение (JPG, PNG, GIF, до 5MB)
3. Увидите предварительный просмотр
4. Нажмите "Загрузить аватар"

### 🔐 Изменение пароля
1. В карточке "Безопасность" введите текущий пароль
2. Введите новый пароль (минимум 6 символов)
3. Подтвердите новый пароль
4. Нажмите "Изменить пароль"

### 🔌 API Endpoints
- `GET /settings` - страница настроек
- `POST /settings/profile` - обновление профиля
- `POST /settings/password` - изменение пароля
- `POST /settings/avatar` - загрузка аватара

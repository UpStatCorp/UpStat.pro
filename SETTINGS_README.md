# Настройки профиля - Новые возможности

## Обзор

Добавлена полнофункциональная система настроек профиля с современным интерфейсом, аналогичным топовым SaaS проектам.

## Новые возможности

### 1. Основная информация
- **Имя пользователя** - редактируемое поле
- **Email** - редактируемое поле с валидацией уникальности
- **Номер телефона** - новое поле для контактной информации

### 2. Фото профиля (Аватар)
- Загрузка изображений (JPG, PNG, GIF)
- Максимальный размер: 5MB
- Автоматическое предварительное просмотр
- Уникальные имена файлов для безопасности
- Автоматическое удаление старых аватаров

### 3. Безопасность
- Изменение пароля с подтверждением
- Проверка текущего пароля
- Валидация нового пароля (минимум 6 символов)

### 4. Дополнительные поля
- **updated_at** - автоматическое обновление времени последнего изменения
- **phone** - номер телефона пользователя
- **avatar** - URL аватара пользователя

## Техническая реализация

### Модель данных
```python
class User(Base):
    id: Mapped[int] = mapped_column(Integer, primary_key=True, index=True)
    email: Mapped[str] = mapped_column(String(255), unique=True, index=True, nullable=False)
    password_hash: Mapped[str] = mapped_column(String(255), nullable=False)
    name: Mapped[str] = mapped_column(String(120), nullable=False)
    phone: Mapped[Optional[str]] = mapped_column(String(20), nullable=True)
    avatar: Mapped[Optional[str]] = mapped_column(String(512), nullable=True)
    created_at: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow)
    updated_at: Mapped[Optional[str]] = mapped_column(String, nullable=True)
```

### API Endpoints
- `GET /settings` - страница настроек
- `POST /settings/profile` - обновление основной информации
- `POST /settings/password` - изменение пароля
- `POST /settings/avatar` - загрузка аватара

### Файловая структура
```
app/
├── routers/
│   └── settings.py          # Роутер настроек
├── templates/
│   └── settings.html        # HTML шаблон настроек
├── static/
│   ├── styles.css           # CSS стили
│   └── avatars/            # Директория для аватаров
└── models.py                # Обновленная модель User
```

## Использование

### Для пользователей
1. Перейдите в раздел "Настройки" в боковом меню
2. Измените основную информацию в карточке "Основная информация"
3. Загрузите аватар в карточке "Фото профиля"
4. Измените пароль в карточке "Безопасность"

### Для разработчиков
1. Все изменения автоматически сохраняются в базу данных
2. Поле `updated_at` обновляется при каждом изменении
3. Аватары сохраняются в `app/static/avatars/`
4. Используйте `user.avatar` для отображения аватара

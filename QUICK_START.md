## 🔧 Если что-то не работает

### Чат скрыт за участниками?
```bash
# Шаг 1: Очистить кеш браузера
Ctrl+Shift+R (Windows/Linux)
Cmd+Shift+R (Mac)

# Шаг 2: Проверить масштаб
Ctrl+0 для сброса на 100%

# Шаг 3: Перезагрузить страницу
F5 или Ctrl+R
```

### Элементы не адаптируются?
```bash
# Проверьте что файл CSS загружен
1. F12 (DevTools)
2. Network tab
3. Найти voice-training.css
4. Должен быть 200 OK, ~70-80KB

# Если 404 Not Found:
- Перезапустить сервер
- Проверить путь к файлу
```

### На мобильном всё мелкое?
```bash
# Проверьте viewport meta tag
В _layout_dashboard.html должно быть:
<meta name="viewport" content="width=device-width, initial-scale=1"/>
```

## 🌟 Ключевые улучшения

### 1. Z-Index исправлен ✨
```css
/* Теперь правильная иерархия */
.training-chat { z-index: 1; }
.training-participants { z-index: 1; }
.training-controls { z-index: 2; }
.checklist-sidebar { z-index: 1000; }
.modal-overlay { z-index: 2000; }
.notifications-container { z-index: 3000; }
```

### 2. Адаптивный Grid ✨
```css
/* Гибкая колоночная система */
.training-main {
  grid-template-columns: minmax(0, 1fr) minmax(280px, 340px);
  /* ↑ Чат растягивается, участники фиксированы */
}
```

### 3. Touch оптимизация ✨
```css
/* Минимум 44px для кнопок (Apple HIG) */
@media (hover: none) and (pointer: coarse) {
  .mic-button, .control-btn {
    min-height: 44px;
    min-width: 44px;
  }
}
```

### 4. Производительность ✨
```css
/* Оптимизация анимаций */
.message-bubble, .mic-button {
  will-change: transform;
  backface-visibility: hidden;
}
```

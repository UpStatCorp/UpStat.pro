# 🎙️ Руководство по настройке стабильности голоса

## Обзор

Голосовой ассистент теперь поддерживает **точную настройку стабильности и тональности голоса** для обеспечения постоянного и предсказуемого звучания.

## 🎯 Параметры стабильности

### ElevenLabs TTS

#### 1. **ELEVENLABS_STABILITY** (Стабильность голоса)
- **Диапазон**: 0.0 - 1.0
- **По умолчанию**: 0.75
- **Описание**: Контролирует постоянство голоса между запросами

```bash
ELEVENLABS_STABILITY=0.75  # Рекомендуется для стабильного голоса
```

**Рекомендации**:
- `0.0-0.3` - Очень экспрессивный, изменчивый голос
- `0.4-0.6` - Сбалансированный (эмоциональный, но управляемый)
- `0.7-0.9` - Стабильный, предсказуемый голос ✅ **Рекомендуется**
- `1.0` - Максимально стабильный (может звучать монотонно)

#### 2. **ELEVENLABS_SIMILARITY_BOOST** (Сходство с оригиналом)
- **Диапазон**: 0.0 - 1.0
- **По умолчанию**: 0.8
- **Описание**: Насколько точно голос повторяет оригинальный голос

```bash
ELEVENLABS_SIMILARITY_BOOST=0.8  # Высокая точность
```

**Рекомендации**:
- `0.0-0.4` - Более креативная интерпретация
- `0.5-0.7` - Сбалансированное сходство
- `0.8-1.0` - Максимальное сходство с оригиналом ✅ **Рекомендуется**

#### 3. **ELEVENLABS_STYLE** (Экспрессивность)
- **Диапазон**: 0.0 - 1.0
- **По умолчанию**: 0.0
- **Описание**: Уровень эмоциональности и выразительности

```bash
ELEVENLABS_STYLE=0.0  # Нейтральная речь без эмоций
```

**Рекомендации**:
- `0.0` - Полностью нейтральная речь ✅ **Рекомендуется для стабильности**
- `0.1-0.3` - Легкая экспрессивность
- `0.4-0.7` - Умеренная эмоциональность
- `0.8-1.0` - Высокая экспрессивность

#### 4. **ELEVENLABS_USE_SPEAKER_BOOST** (Усиление четкости)
- **Тип**: Boolean (true/false)
- **По умолчанию**: true
- **Описание**: Улучшает четкость и разборчивость речи

```bash
ELEVENLABS_USE_SPEAKER_BOOST=true  # Включено для лучшей четкости
```

**Рекомендации**:
- `true` - Улучшенная четкость ✅ **Рекомендуется**
- `false` - Естественное звучание

### OpenAI TTS

#### **OPENAI_TTS_SPEED** (Скорость речи)
- **Диапазон**: 0.25 - 4.0
- **По умолчанию**: 1.0
- **Описание**: Скорость воспроизведения речи

```bash
OPENAI_TTS_SPEED=1.0  # Нормальная скорость
```

**Рекомендации**:
- `0.5-0.8` - Медленная, размеренная речь
- `1.0` - Нормальная скорость ✅ **Рекомендуется для стабильности**
- `1.1-1.3` - Немного быстрее
- `1.4-2.0` - Быстрая речь

## 🎛️ Рекомендуемые конфигурации

### Для максимальной стабильности (рекомендуется)

```bash
# ElevenLabs - стабильный профессиональный голос
ELEVENLABS_STABILITY=0.75
ELEVENLABS_SIMILARITY_BOOST=0.8
ELEVENLABS_STYLE=0.0
ELEVENLABS_USE_SPEAKER_BOOST=true

# OpenAI - стабильная скорость
OPENAI_TTS_SPEED=1.0
```

**Эффект**: Постоянный, предсказуемый голос с нейтральной интонацией

### Для естественного разговора

```bash
# ElevenLabs - естественный голос с легкими вариациями
ELEVENLABS_STABILITY=0.6
ELEVENLABS_SIMILARITY_BOOST=0.75
ELEVENLABS_STYLE=0.2
ELEVENLABS_USE_SPEAKER_BOOST=true

# OpenAI

```

**Эффект**: Более естественное звучание с легкими эмоциональными нюансами

### Для экспрессивного ассистента

```bash
# ElevenLabs - экспрессивный голос
ELEVENLABS_STABILITY=0.5
ELEVENLABS_SIMILARITY_BOOST=0.7
ELEVENLABS_STYLE=0.5

# OpenAI

```

**Эффект**: Эмоциональный, живой голос с вариациями

### Для монотонного диктора

```bash
# ElevenLabs - максимально ровный голос
ELEVENLABS_STABILITY=0.9
ELEVENLABS_SIMILARITY_BOOST=0.85
ELEVENLABS_STYLE=0.0
ELEVENLABS_USE_SPEAKER_BOOST=true

```

**Эффект**: Очень ровный, предсказуемый голос без вариаций

## 🔧 Как применить настройки

1. **Откройте файл `.env`**:
```bash
nano .env
```

2. **Добавьте или измените параметры**:
```bash
# Копируйте нужную конфигурацию выше
ELEVENLABS_STABILITY=0.75
ELEVENLABS_SIMILARITY_BOOST=0.8
# и т.д.
```

3. **Перезапустите голосовой ассистент**:
```bash
# Docker
docker-compose restart voice_assistant

# Или напрямую
cd voice_assistant
python -m uvicorn main:app --reload
```

## 📊 Мониторинг параметров

При запуске ассистента вы увидите в логах текущие параметры:

### Для выразительного голоса:

## 💡 Советы

1. **Начните с рекомендуемых значений** и постепенно настраивайте
2. **Изменяйте по одному параметру** за раз для понимания эффекта
3. **Высокая стабильность** (0.7+) критична для постоянного голоса
4. **STYLE=0.0** убирает эмоциональные колебания
5. **SPEAKER_BOOST** всегда держите включенным для четкости

## 🐛 Решение проблем

### Проблема: Голос звучит по-разному каждый раз

**Решение**: Увеличьте параметры стабильности:
```bash
ELEVENLABS_STABILITY=0.8
ELEVENLABS_SIMILARITY_BOOST=0.85
ELEVENLABS_STYLE=0.0
```

### Проблема: Голос звучит слишком монотонно

**Решение**: Немного снизьте стабильность или добавьте стиль:
```bash
ELEVENLABS_STABILITY=0.65
ELEVENLABS_STYLE=0.15
```

### Проблема: Речь недостаточно разборчива

**Решение**: Убедитесь, что speaker_boost включен:
```bash
ELEVENLABS_USE_SPEAKER_BOOST=true
```

### Проблема: Тональность прыгает

**Решение**: Максимизируйте стабильность и сходство:
```bash
ELEVENLABS_STABILITY=0.85
ELEVENLABS_SIMILARITY_BOOST=0.9
ELEVENLABS_STYLE=0.0
```

## 📈 Влияние на производительность

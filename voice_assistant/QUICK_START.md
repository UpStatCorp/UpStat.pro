## 🎯 Как запустить

### Шаг 1: Проверьте конфигурацию

Убедитесь, что у вас есть файл `.env` (если нет, скопируйте из `env.example`):

```bash
cp env.example .env
```

### Шаг 2: Настройте API ключи в `.env`

```bash
# Обязательно для работы:
OPENAI_API_KEY=ваш_ключ_openai

# Опционально для ElevenLabs TTS:
ELEVENLABS_API_KEY=ваш_ключ_elevenlabs
ELEVENLABS_VOICE_ID=ваш_voice_id

# Рекомендуемые настройки для максимальной скорости:
STT_PROVIDER=whisper_openai
TTS_PROVIDER=openai
TTS_MODEL=tts-1
TTS_VOICE=nova
GPT_MODEL=gpt-4o-mini
GPT_MAX_TOKENS=150

# Параметры стабильности голоса (для постоянной тональности):
ELEVENLABS_STABILITY=0.75
ELEVENLABS_SIMILARITY_BOOST=0.8
ELEVENLABS_STYLE=0.0
ELEVENLABS_USE_SPEAKER_BOOST=true
OPENAI_TTS_SPEED=1.0
```

### Шаг 3: Запустите ассистента

```bash
cd voice_assistant
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Или через Docker:
```bash
docker-compose up voice_assistant
```

### Шаг 4: Откройте в браузере

```
http://localhost:8000
```

## 🔧 Тонкая настройка

### Если ассистент прерывает вас

В `voice_assistant/config.py`:
```python
SILENCE_DURATION_MS = 700  # Увеличьте до 700-800мс
```

В `voice_assistant/router.py`:
```python
SILENCE_THRESHOLD = 0.7  # Увеличьте до 0.7-0.8
```

### Если хотите ещё быстрее

В `.env`:
```bash
STT_PROVIDER=whisper_openai  # Самый быстрый
TTS_PROVIDER=openai          # Быстрее чем elevenlabs
TTS_MODEL=tts-1              # НЕ tts-1-hd
GPT_MAX_TOKENS=100           # Ещё короче
```

### Если нужно больше качества

В `.env`:
```bash
STT_PROVIDER=elevenlabs      # Самый точный STT
TTS_PROVIDER=elevenlabs      # Самый качественный TTS
ELEVENLABS_MODEL_ID=eleven_multilingual_v2
GPT_MODEL=gpt-4o             # Умнее, но медленнее
GPT_MAX_TOKENS=200           # Более развёрнутые ответы
```

## 🐛 Решение проблем

### Проблема: "Voice assistant modules not available"
**Решение**: Проверьте, что установлены все зависимости:
```bash
pip install -r requirements.txt
```

### Проблема: "OPENAI_API_KEY не установлен"
**Решение**: Добавьте ключ в `.env`:
```bash
OPENAI_API_KEY=sk-...
```

### Проблема: Медленный ответ
**Решение**: 
1. Проверьте скорость интернета
2. Используйте `STT_PROVIDER=whisper_openai`
3. Используйте `TTS_PROVIDER=openai`
4. Уменьшите `GPT_MAX_TOKENS=100`

### Проблема: Ассистент прерывает речь
**Решение**: Увеличьте паузу детекции до 700-800мс

### Проблема: Голос звучит по-разному каждый раз
**Решение**: 
1. Проверьте параметры стабильности в `.env`
2. Увеличьте `ELEVENLABS_STABILITY=0.8`
3. Убедитесь что `ELEVENLABS_STYLE=0.0`
4. См. `VOICE_STABILITY_GUIDE.md` для подробностей

## 🚀 Как начать использовать

1. **Если у вас ещё нет `.env` файла**, скопируйте из примера:
   ```bash
   cp env.example .env
   ```

2. **Настройте API ключи** в `.env`:
   ```bash
   OPENAI_API_KEY=ваш_ключ
   ELEVENLABS_API_KEY=ваш_ключ  # опционально
   ```

3. **Убедитесь, что установлены правильные провайдеры** в `.env`:
   ```bash
   STT_PROVIDER=whisper_openai
   TTS_PROVIDER=elevenlabs  # или openai для ещё большей скорости
   GPT_MODEL=gpt-4o-mini
   ```

4. **Перезапустите голосовой ассистент**:
   ```bash
   # Если используете Docker
   docker-compose restart voice_assistant
   
   # Или если запускаете напрямую
   cd voice_assistant
   python -m uvicorn main:app --reload
   ```

## ⚠️ Важные примечания

- **Слишком короткая пауза** (< 500мс) может привести к преждевременной обработке
- **Рекомендуемый диапазон паузы**: 600-800мс
- **Текущая настройка**: 650мс - оптимальный баланс скорости и точности

## 🐛 Если возникли проблемы

1. **Ассистент прерывает вас во время речи**:
   - Увеличьте `SILENCE_DURATION_MS` до 700-800
   - Увеличьте `SILENCE_THRESHOLD` до 0.7-0.8

2. **Ассистент слишком медленно реагирует**:
   - Убедитесь, что используете `STT_PROVIDER=whisper_openai`
   - Проверьте скорость интернета
   - Рассмотрите использование `TTS_PROVIDER=openai` вместо elevenlabs

3. **Ассистент не распознаёт речь**:
   - Проверьте качество микрофона
   - Уменьшите `SPEECH_THRESHOLD` в `router.py` до 0.015
   - Убедитесь, что API ключи корректны

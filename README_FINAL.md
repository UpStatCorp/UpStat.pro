## 🚀 Как запустить

### **1. Запуск приложения**
```bash
cd app
python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### **2. Открыть в браузере**
- **Дашборд**: http://localhost:8000/dashboard
- **Настройки**: http://localhost:8000/settings
- **Чат**: http://localhost:8000/chat
- **История**: http://localhost:8000/calls

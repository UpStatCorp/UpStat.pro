## 💡 Интерактивные элементы

### Hover эффекты (Desktop/Laptop)
```css
Кнопка:
  Default:     [  Кнопка  ]
  Hover:       [▲ Кнопка ▲] ← поднимается
  Active:      [▼ Кнопка ▼] ← нажата

Микрофон:
  Default:     ( 🎙️ ) + тень
  Hover:       ( 🎙️ ) + больше + ярче
  Active:      ( 🎙️ ) + пульсация
```

### Touch эффекты (Mobile/Tablet)
```css
Tap:
  Start:    [  Кнопка  ]
  Active:   [▼ Кнопка ▼] (немедленно)
  End:      [  Кнопка  ] + действие
```

## 🎭 Анимации

### 1. Пульсация микрофона (активный)
```
Цикл 2s:
  0%:   scale(1.0) + тень нормальная
  50%:  scale(1.03) + тень увеличена + glow
  100%: scale(1.0) + тень нормальная
```

### 2. Печатание ИИ
```
3 точки:
  ● (delay 0s)   ↕️ прыгает
  ● (delay 0.2s) ↕️ прыгает
  ● (delay 0.4s) ↕️ прыгает
```

### 3. Звуковые волны (ИИ говорит)
```
5 линий:
  ▂ ▃ ▅ ▇ ▅  ← волна
  ▃ ▅ ▇ ▅ ▃  ← анимация
  ▅ ▇ ▅ ▃ ▂  ← слева направо
```

### 4. Появление сообщений
```
Новое сообщение:
  0%:   opacity: 0, translateY(10px)
  100%: opacity: 1, translateY(0)
  Duration: 0.3s ease
```

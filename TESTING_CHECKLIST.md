# ✅ Чеклист тестирования - Голосовая тренировка

## 🖥️ Desktop тестирование

### Разрешения
- [ ] **1920x1080** - Полный HD
  - [ ] Чат слева (~70% ширины)
  - [ ] Участники справа (~30% ширины)
  - [ ] Микрофон 140x140px
  - [ ] Все элементы видны без скролла

- [ ] **1440x900** - MacBook Pro
  - [ ] Layout корректный
  - [ ] Панель участников 300-340px
  - [ ] Нет горизонтального скролла

- [ ] **1366x768** - Стандартный ноутбук
  - [ ] Компактная раскладка
  - [ ] Панель участников 260-300px
  - [ ] Прокрутка работает плавно

### Масштаб браузера
- [ ] **50%** - Всё читабельно
- [ ] **75%** - Нормальное отображение
- [ ] **100%** - Идеально ✨
- [ ] **125%** - Правильно масштабируется
- [ ] **150%** - Элементы не наложены
- [ ] **200%** - Работает корректно

### Функциональность
- [ ] Чат не скрывается за участниками
- [ ] Кнопки не перекрывают текст
- [ ] Hover эффекты работают
- [ ] Прокрутка чата плавная
- [ ] Прокрутка участников работает
